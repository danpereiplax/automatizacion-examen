# Automatización de Pruebas – Examen Final  
Proyecto base para la Actividad 1: configuración de Git, Maven y flujo de ramas (Trunk-Based).

---

 1. Descripción del proyecto

Este proyecto corresponde a la Actividad 1 del examen final de la asignatura **Automatización de Pruebas**.  
Su objetivo es crear un entorno profesional inicial para pruebas automatizadas en Java, utilizando:

- **Git** para control de versiones  
- **Trunk-Based Development** como flujo de ramas  
- **Maven** para administración de dependencias  
- **JUnit 5** como framework de pruebas  
- **Selenium** para pruebas UI automatizadas (dependencia ya incluida)

El repositorio contiene la estructura estándar Maven y un primer test automatizado, siguiendo las buenas prácticas indicadas en el material ME y en las Actividades Formativas.
